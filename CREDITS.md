# Изображения

Все снимки на лендинге **сгенерированы нейросетью**, а не сняты фотографом.
Модель — Stable Diffusion XL (`dreamshaperXL_lightning`), локальный Forge/A1111.
Скрипт с промптами и сидами: [`scripts/generate-photos.mjs`](scripts/generate-photos.mjs).
Набор воспроизводим: сид у каждого кадра фиксирован.

Ни одна строка ниже не приписывает авторство живому человеку. Портрет автора
курса — тоже сгенерированное изображение вымышленного человека; курс, автор и
работы выпускников вымышлены, о чём сказано в футере и в README.

| Файл                     | Размер    | Что на снимке                       | Сид                                          |
| ------------------------ | --------- | ----------------------------------- | -------------------------------------------- |
| `src/images/hero.webp`   | 1600×1067 | Проявочная под красным фонарём      | 34                                           |
| `src/images/author.webp` | 900×1200  | Портрет автора под красным фонарём  | 7002                                         |
| `src/images/works/01…12` | 900×600   | 12 кадров контрольного листа, 3 : 2 | 7100–7103, 91, 7105–7106, 214, 7108–7110, 82 |

Сиды кадров `05`, `08` и `12` выпадают из ряда `7100 + N`: на исходных числах
модель рисовала занавеску без ветра, трамвай с маршрутным табло и сухой
отпечаток в пустой кювете, то есть не то, что обещает `alt`. Удачные сиды лежат
не подряд, и подобранный кадр важнее красивой нумерации.

## Ни рук, ни надписей

Гнутые пальцы и нечитаемая псевдотипографика — два признака, по которым
генерацию узнают с первого взгляда. Стоп-слова их не убирают: при `cfg_scale: 2`
негатив почти не работает. Поэтому сюжеты выбраны так, чтобы модели просто
нечего было писать и не за что держаться:

| Кадр     | Было                           | Стало                            |
| -------- | ------------------------------ | -------------------------------- |
| `04`     | Руки с катушкой плёнки         | Капли дождя на стекле            |
| `08`     | Трамвай с маршрутным табло     | Пустые рельсы в снегу            |
| `09`     | Банки с этикетками             | Гладкие бачки без надписей       |
| `author` | Портрет по пояс, руки на столе | Портрет по грудь, руки вне кадра |

Промпты кадров контрольного листа написаны под `alt`-тексты из
[`src/content/works.ts`](src/content/works.ts): описание для скринридера должно
совпадать с тем, что на снимке действительно есть, иначе оно врёт. Меняется
снимок — меняется и `alt`.

Плёнки в подписях только чёрно-белые (Tri-X, HP5, Delta): курс печатает руками
на баритовой бумаге, цветную C-41 в этой лаборатории не проявляют. Кадры
контрольного листа сгенерированы монохромными, герой и портрет — в красном
свете фонаря.

## Как перегенерировать

```bash
SD_URL=http://<forge-host>:7860 node scripts/generate-photos.mjs
ONLY=hero  SD_URL=… node scripts/generate-photos.mjs   # одна цель
```

Forge должен быть запущен с `--api`. Скрипт складывает WebP по тем же путям,
что ждёт вёрстка, и падает, если суммарный вес превысил **1.8 МБ**.

## Если понадобятся настоящие фотографии

Подойдут [Unsplash](https://unsplash.com) и [Pexels](https://pexels.com) по
запросам `film photography`, `darkroom`, `analog` — лицензии допускают
использование, в том числе коммерческое. Тогда эту таблицу нужно заменить на
список вида:

```
| `src/images/works/01.webp` | Имя Фамилия | https://unsplash.com/photos/XXXXXXX |
```

Размеры и пропорции менять нельзя: кадры контрольного листа строго 3 : 2, иначе
сетка поедет.
